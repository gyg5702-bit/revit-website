어떻게 발주할까요?

Revit 없이 바로 주문하거나,
BIM 데이터를 연동해 자동으로 발주할 수 있습니다.

🛒
MODE 01 — 직접 발주
원하는 자재를
직접 주문하기

Revit 없이도 가능합니다. 자재명과 수량을 직접 입력하면 AI가 최적 업체를 추천하고 발주서를 생성합니다.

시작하기
🏗️
MODE 02 — BIM 연동
Revit 데이터로
자동 발주하기

Revit 애드인이 서버로 전송한 물량 데이터를 불러옵니다. LLM이 자재를 분류하고 업체를 자동 추천합니다.

데이터 불러오기
🤖
MODE 03 — AI 어시스턴트
Claude AI에게
자연어로 물어보기

"D19 철근 물량 알려줘", "발주 가능한 자재 비교해줘" 같은 자연스러운 한국어 질문에 AI가 BIM 데이터를 보고 답변합니다.

채팅 시작
🏛️
MODE 04 — 3D 뷰어
IFC 모델
3D로 살펴보기

서버의 IFC 파일을 브라우저에서 직접 렌더링. 부재 클릭 시 발주 정보·좌표가 즉시 표시됩니다. 카테고리별 자동 색상, 검색, 층 필터 지원.

모델 열기
📤
MODE 05 — BIM 업로드
IFC 파일에서
자재 자동 추출

Revit·BuildingCheck 없이도 IFC 파일만 드래그하면 브라우저가 직접 파싱해 자재·층·좌표를 추출하고, 서버에 envelope JSON·IFC 한 번에 업로드합니다.

// MODE 02 · 2단계 — 자재·층 선택

자재와 층을 선택하세요

JSON에서 읽은 물량을 자재별로 확인하고, 발주할 층을 지정하세요.

자재 선택 — 자재마다 층을 따로 지정 가능
옵션 · 안내
■ 자재마다 우측에서 층을 개별 지정
■ (전체) = 그 자재의 모든 층 합산
■ 체크한 자재만 업체 추천으로 넘어갑니다
■ JSON에 물량이 없는 자재는 비활성화됩니다
■ 철근·조적은 발주내역, 콘크리트·거푸집은 벽체 물량 기준
추출 결과 미리보기
자재층수량단위
자재를 체크하면 추출 물량이 표시됩니다.
